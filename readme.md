# Static Website on AWS S3

## Project Overview
A static website hosted on Amazon S3 with CloudFront distribution for fast content delivery.

## Technologies Used
- AWS S3 (Simple Storage Service)
- AWS CloudFront (CDN)
- HTML/CSS

## Features
- Low-cost hosting (~$0.50/month)
- Global content delivery
- High availability (99.99% uptime)

## Deployment Steps
1. Create S3 bucket
2. Enable static website hosting
3. Upload files
4. Configure CloudFront
5. Set up Route 53 (optional)

## Project Benefits
- Learn AWS fundamentals
- Understand content delivery
- Cost-effective hosting solution
